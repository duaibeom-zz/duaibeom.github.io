CUDA Installations

Requirements

  • Nvidia driver (latest version)

CUDA Installation guide (Multiple version; Ubuntu >= 18.04)

  • CUDA Installation Guide
# Ex. CUDA 11.1
# Do not insert in apt preference. It makes it impossible to install various versions of CUDA.
#
# wget https://developer.download.nvidia.com/compute/cuda/repos/ubuntu2004/x86_64/cuda-ubuntu2004.pin
# sudo mv cuda-ubuntu2004.pin /etc/apt/preferences.d/cuda-repository-pin-600

wget https://developer.download.nvidia.com/compute/cuda/11.1.1/local_installers/cuda-repo-ubuntu2004-11-1-local_11.1.1-455.32.00-1_amd64.deb
sudo dpkg -i cuda-repo-ubuntu2004-11-1-local_11.1.1-455.32.00-1_amd64.deb
sudo apt-key add /var/cuda-repo-ubuntu2004-11-1-local/7fa2af80.pub
sudo apt-get update

sudo apt-get -y install cuda-libraries-11-1  # It contains only runtime libraries.
# sudo apt-get -y install cuda-toolkit-11-1  # It does not contain a graphic driver.

cuDNN Installation guide

  • cuDNN Installation Guide

Recommand Tar File installation (Because, It does not depend on OS)

CUDA_VERSION=10.2  # tmp
tar xzvf cudnn-$CUDA_VERSION-linux-x64-v8.x.x.x.tgz  # Will be extraced in `cuda` dir.

sudo cp cuda/include/cudnn*.h /usr/local/cuda-$CUDA_VERSION/include
sudo cp cuda/lib64/libcudnn* /usr/local/cuda-$CUDA_VERSION/lib64

sudo chmod a+r /usr/local/cuda-$CUDA_VERSION/include/cudnn*.h /usr/local/cuda-$CUDA_VERSION/lib64/libcudnn*
