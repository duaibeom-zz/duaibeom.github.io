Tmux 사용법

새로운 세션 생성: tmux new (-s session_name)

세션 종료: exit

세션 목록: tmux ls

세션 중단: ^b d

세션 다시시작: tmux attach (-t session id or name)

새 윈도우 생성: ^b c

윈도우 이동: ^b b # (이동하고자 하는 번호 입력)
