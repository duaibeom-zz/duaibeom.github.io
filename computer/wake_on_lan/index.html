원격 부팅 (Wake On LAN)

개인 PC 원격 부팅

원격 접속을 하기 위해서는 다음 4가지가 필요하다.

  1. Host 컴퓨터에 WOL 설정이 되어 있어야 한다. https://www.techrepublic.com/article/how-to-enable-wake-on-lan-in-ubuntu-server-18-04/
  2. Host 컴퓨터 메인보드에 해당 기능을 가능하게 해야한다. https://www.howtogeek.com/70374/how-to-geek-explains-what-is-wake-on-lan-and-how-do-i-enable-it/
  3. Router(공유기)에 해당 컴퓨터를 공유기 WOL 기능에 추가한다. https://m.blog.naver.com/junho306/221397846945
  4. App 혹은 공유기 주소 설정이 되어 있어야 한다.

서버 원격 부팅

우리 연구실 서버는 WOL을 사용할 필요가 없다. IPMI 기능이 있기 때문에 이것을 이용하면 된다.

  1. IPMI로 접속한다. via ssh; (자세한 사항은 구두를 통해 전달을 받으세요.)
  2. cd system1
  3. cd pwrmgtsvc1
  4. show
  5. start 서버 노드 부팅이 시작된다.
  6. show Power state가 6에서 1로 변경된 것을 볼 수 있다.
  7. exit
  8. 귀찮다. 확인이고 뭐고 한번에 하자. start /system1/pwrmgtsvc1
