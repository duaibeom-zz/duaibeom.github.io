원격 부팅 (Wake On LAN)

원격 부팅 (Wake On LAN) 개인 PC 원격 부팅 원격 접속을 하기 위해서는 다음 4가지가 필요하다. Host 컴퓨터에 WOL 설정이 되어 있어야 한다. https://www.techrepublic.com/article/how-to-enable-wake-on-lan-in-ubuntu-server-18-04/ Host 컴퓨터 메인보드에 해당 기능을 가능하게 해야한다. https://www.howtogeek.com/70374/how-to-geek-explains-what-is-wake-on-lan-and-how-do-i-enable-it/ Router(공유기)에 해당 컴퓨터를 공유기 WOL 기능에 추가한다. https://m.blog.naver.com/junho306/221397846945 App 혹은 공유기 주소 설정이 되어 있어야 한다. 서버 원격 부팅 우리 연구실 서버는 WOL을 사용할 필요가 없다.
