Jupyter Notebook

Posted at — May 13, 2020

Jupyter notebook Sever

  1. via ssh
  2. via config

via config

  • config 파일 생성 jupyter notebook --generate-config

  • password 설정

jupyter notebook password
Enter password:  ****
Verify password: ****
[NotebookPasswordApp] Wrote hashed password to /Users/you/.jupyter/jupyter_notebook_config.json

jupyter_notebook_config.json에 있는 hash password 를 /Users/you/.jupyter/jupyter_notebook_config.py에 넣기.

c.NotebookApp.password = u'sha1: ~ '

  • config 파일 설정

c.NotebookApp.allow_origin = '192.168.0.0' 접속가능한 IP 설정.

c.NotebookApp.ip = '192.168.0.???' 해당 노드의 IP 입력.

c.NotebookApp.notebook_dir = '/home/User/Work/dir' Working dir 설정

c.NotebookApp.open_browser = False
